# About OpenZone Rustaceans

## What is it?

OpenZone Rustaceans is a **desktop AI assistant** project built from a Rust-native foundation. The goal is to connect AI model capabilities with desktop context and workflows so users can complete work directly from their machine.

## The Problem

Most AI assistants still live in separate browser tabs or chat windows. They can answer questions, but they are often disconnected from local files, apps, and the work happening on the desktop.

## The Solution

OpenZone Rustaceans aims to provide a native assistant layer with:

- Fast and reliable Rust-powered application logic
- Desktop workflow integration for practical task assistance
- AI model connector architecture for cloud and/or local providers
- User-controlled context sharing and privacy-aware defaults
- Extensible tools for automation, planning, writing, and execution

## Vision

A desktop companion that understands user intent, safely uses local context, integrates with AI models, and helps people finish real work faster while keeping them in control.

## Status

🚧 Early development. The repository currently contains a single Cargo package with internal `onboarding` and `shared` modules, plus an Iced-based first-run onboarding prototype. AI integrations and broader desktop workflows are planned next. See the [roadmap](README.md#️-roadmap) and [architecture notes](docs/ARCHITECTURE.md).
